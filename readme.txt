Olá!
Esta versão atualizada é compatível com XML.
Se você tiver problemas com esta versão desde o seu PHP não tem
foram atualizados, use o arquivo hangman_old.php que é o anterior
versão do meu jogo.
No momento em que o arquivo que contém as palavras é que parole.txt
está incluído.
Você pode alterar todas as palavras neste arquivo.

Agora Hangman analisa um arquivo de modelo para que você possa facilmente integrar o meu jogo
com o seu site.
Basta levar o seu modelo (qualquer página padrão do seu site), editá-lo
com o seu programa favorito (ou seja, bloco de notas) e adicione a linha
<-! HANGMAN ->
entre duas linhas em branco onde você deseja inserir Hangman.
Salve este arquivo com o nome base.html para o diretório onde você
colocar Hangman. Hangman irá ler este arquivo e analisá-lo com o jogo!
